N) ShortCodes - top

Showtime WordPress template is fully compatible with popular shortcodes:

  1. [one_half] [/one_half]
  2. [one_half_last] [/one_half_last]
  3. [one_third] [/one_third]
  4. [one_third_last] [/one_third_last]
  5. [one_fourth] [/one_fourth]
  6. [one_fourth_last] [/one_fourth_last]
  7. [two_third] [/two_third]
  8. [two_third_last] [/two_third_last]
  9. [three_fourth] [/three_fourth]
  10. [three_fourth_last] [/three_fourth_last]

You can use these shortcodes to divide layout into columns. Please, do not forgot to use as a last shortcode a shortcode with added _last. Its because of CSS layout clearing. For DEMO please visit the Live Preview from Item page.

You can create this effect in footer widgets by adding shortcode [first] [/first] into footer widget title like this:

[first]This is a first line[/first] This is a second line
